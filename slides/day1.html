พื้นฐาน

TouchDesigner คืออะไร

เครื่องมือ real-time node-based dataflow — เราวาง "สาย" ของการแปลงข้อมูลไว้ แล้วโปรแกรมก็คำนวณภาพ/เสียงให้สดทุกเฟรม

นึกภาพเหมือน สายพานโรงงาน: ข้อมูลดิบไหลเข้าทางซ้าย ผ่านเครื่องแปลงทีละตัว ออกมาเป็นภาพทางขวา — เราออกแบบว่าจะมีเครื่องอะไรบ้าง ต่อกันยังไง

ไม่ได้เขียนโปรแกรมทีละบรรทัด — แต่ "วางสาย" แล้วโปรแกรมรันให้
พื้นฐาน

TD ใช้ภาษา / เทคโนโลยีอะไร

  • ตัว engine เขียนด้วย C++ รันบน GPU — งานหนัก (ภาพ, signal, 3D, render) อยู่ชั้นนี้ เร็วมาก
  • Python ฝังมาในตัว — ไว้เขียน logic, ควบคุม, เชื่อมระบบ. ทำงานส่วนใหญ่ได้โดยไม่ต้องเขียน Python แค่ต่อ node
  • GLSL — เขียน shader บน GPU เองได้ ถ้าอยากลงลึก
  • ต่อ custom operator ด้วย C++ ได้ ถ้าต้องการของที่ไม่มีในตัว
เริ่มต้นแค่ต่อ node + กรอกค่า · งานจริงจังค่อยหยิบ Python มาช่วย

พื้นฐาน

Cook — คำนวณเมื่อจำเป็นเท่านั้น

node จะ "cook" (คำนวณใหม่) ก็ต่อเมื่อ มีคนปลายทางขอใช้ค่ามัน และ มีอะไรเปลี่ยน เท่านั้น

  • กิ่งที่ไม่มีใครดู ไม่มีใครใช้ = ไม่คำนวณ = ฟรี ไม่กินเครื่อง
  • เปิด viewer ดู node ไหน = สั่งให้มันคำนวณ → ปิด viewer ที่ไม่ใช้ช่วยให้ลื่นขึ้น
เหมือน spreadsheet — แก้ช่องนึง มันคิดใหม่เฉพาะช่องที่เกี่ยวข้อง ไม่คิดทั้งแผ่น
ต่อค่าข้าม node

Reference vs Export

Reference (Expression)

  • พิมพ์สูตรในช่อง parameter
  • ใส่สูตร/คณิต/เงื่อนไขได้
  • ดึงจากอะไรก็ได้ (CHOP, param, เวลา)

op('lfo1')['chan1'] * 2

Export

  • ลาก CHOP channel มาวางทับ parameter
  • ใส่สูตรไม่ได้ — ส่งค่าดิบตรงๆ
  • รับจาก CHOP channel เท่านั้น

ลากเร็วดี เวลาไม่ต้องแต่งค่า

จำง่ายๆ: ต้องแปลงค่า → Reference · เอาตรงๆ → Export · ทำ UI → Bind (สองทาง)
ต่อค่าข้าม node

Logic ใน TD — ไม่มีปุ่ม if

TD เป็นสายข้อมูลที่ไหลตลอด เลยไม่ "กระโดดข้ามบรรทัด" แบบ if/else — แต่ใช้วิธี เลือกว่าเส้นไหนจะไหลผ่าน แทน

  • Switch = เลือก input ตาม index (เหมือนสับราง) — ตัวนี้แทน if/else ได้บ่อยสุด
  • Logic CHOP = เทียบค่า/threshold → ได้ 0 หรือ 1 (จริง/เท็จ)
  • เงื่อนไขซับซ้อนจริงๆ → เขียน Python ใน Script/Execute
ตัวอย่าง Day 2: "มือโผล่ในเฟรมไหม" → Logic CHOP → Switch สลับภาพ idle/active

Particle (instancing)

หัวใจของงานปลายทาง — วาดทรงเดียวซ้ำเป็นหมื่นตัวในการวาดครั้งเดียว
