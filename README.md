Instructor: **Wave Pongruengkiat** (Creative Tech Studio).

## เริ่มยังไง

1. กด **Code → Download ZIP** (มุมขวาบน) เพื่อโหลดทั้งหมด — หรือเปิดดูทีละไฟล์ก็ได้
2. เปิด `handout.html` ในเบราว์เซอร์ = คู่มือหลักระหว่างเรียน
3. ไฟล์ `.toe` แต่ละบทอยู่ใน `files/day1`, `files/day2`, `files/day3`
