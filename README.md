# TouchDesigner Workshop — Creative Tech Studio × ดีดีเอ็กซ์พี (DDEXP)

3-day hands-on TouchDesigner workshop. **2 · 3 · 5 June 2026**, on-site.
Instructor: **Wave Pongruengkiat** (Creative Tech Studio).

> ที่นี่คือที่รวมไฟล์ทั้งหมดของคอร์ส — curriculum, สไลด์, ตารางเวลา และไฟล์ `.toe` แต่ละ tutorial. โหลดได้เลย ไม่ต้อง login.

## เริ่มยังไง

1. กด **Code → Download ZIP** (มุมขวาบน) เพื่อโหลดทั้งหมด — หรือเปิดดูทีละไฟล์ก็ได้
2. เปิด `curriculum.html` = เนื้อหาหลัก + run-of-show ทุก session · `slides/day1.html` = สไลด์ Day 1
3. ไฟล์ `.toe` แต่ละบทอยู่ใน `files/day1`, `files/day2`, `files/day3`
